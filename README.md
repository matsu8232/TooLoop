# TooLoop（ツーループ）

# 概要
このアプリは、事務用品のシェアをサポートするアプリケーションです。自身の所属が持っている事務用品を登録することができ、アプリ上で事務用品の検索をして、貸出状況の確認や予約・相談を行うことができます。
