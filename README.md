# 概要
このアプリは、所属間の事務用品のシェアをサポートするアプリケーションです。自身の所属が持っている事務用品を登録することができ、アプリ上で事務用品の検索をして、貸出状況の確認や貸出予約・相談を行うことができます。
## サイトテーマ
「資産をループさせることで最大限活用する」
## テーマを選んだ理由

### 【きっかけ】
庶務・予算管理を担当していたときに、他所属の職員から「こちらにコンテナ型のキャリーがあるという噂を聞いて…」と確認と貸出依頼を受けたのがきっかけです。
### 【問題点】
・年に数回しか使わないものを各所属ごとに購入すること余計な支出が発生している  
・どの所属が何を持っているか把握できていない  
・関わりのない所属に対して貸し借りの交渉をするには心理的ハードルが高い
### 【解決策】
・どこが何を持っているのかを見える化する  
・アプリ上で予約できるようにすることで心理的負担を軽減させる
