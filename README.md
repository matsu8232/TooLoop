# 概要
このアプリは、組織内の備品のシェアをサポートするアプリケーションです。各課が持っている備品を登録することができ、アプリ上で必要な備品を検索して、詳細確認や貸出予約・交渉をすることができます。

## サイトテーマ
「資産をループさせて最大限活用する」

## テーマを選んだ理由
備品の購入はお金がかかるだけでなく事務的にも負担になります。組織内でシェアすることで両方の負担を軽減できるのではと考え、シェアサポートアプリを制作しようと思いました。

### 【きっかけ】
課の庶務を担当していたときに、他課の職員から「こちらにコンテナ型のキャリーがあるという噂を聞いて……」と確認と貸し出しをお願いされたのがきっかけです。

### 【問題点】
* 年に数回しか使わないものを各課ごとに購入することで余計な支出と事務負担が発生している  
* どの課が何を持っているか把握できていない  
* 関わりのない課に貸し出し依頼をするのは心理的ハードルが高い

### 【解決策】
* 投稿機能により各課が持っている備品を可視化
* 予約機能により気軽に貸出依頼をすることができるように
* DM機能で細かい交渉も可能

# ターゲットユーザー
各課で備品を購入しているような企業や地方公共団体の庶務担当

# 主な利用シーン
* 一期間だけ備品が必要になった時
* 急な故障で備品が使えなくなった特
* 購入前に備品の性能を試したい時
